# Contributing to boltpy

Thanks for your interest in contributing!

## Ways to contribute
- Report bugs and request features via GitHub Issues
- Improve documentation (README, examples)
- Add tests or new connectors/sources
- Submit pull requests for fixes and improvements

## Getting help / asking questions
If you have a usage question, please open a GitHub Issue and label it as a question or write an email to filippo.civilini@studio.unibo.it
